# ToDo-API-BSBO

## О проекте

ToDo-API-BSBO — это учебный REST API‑сервис для управления задачами (to-do), разработанный в рамках практической работы по дисциплине «Интероперабельность приложений и систем». Проект демонстрирует базовые подходы к проектированию моделей данных, реализации CRUD‑операций и версионированию API. 

## Основной функционал

- Создание новой задачи через POST‑запрос с валидацией входных данных.
- Обновление данных задачи и пометка задачи как завершенной отдельными методами.
- Удаление задачи по идентификатору.
- Получение статистики по задачам через отдельный модуль статистики.

Все роуты API используют общий префикс `/api/v1`, а статистика вынесена в отдельный роутер с префиксом `/stats` для удобства поддержки и версионирования. 

## Документация API

Проект содержит авто‑генерируемую документацию в Swagger, где можно просматривать доступные методы, их параметры и примеры запросов и ответов. Это упрощает тестирование и интеграцию с клиентскими приложениями. 

## Валидация и обработка ошибок

- Обработка некорректного JSON с возвратом корректного кода ошибки.
- Проверка ограничений на поля (например, минимальная длина названия задачи).
- Поддержка создания задач без описания, если это допускается бизнес‑логикой.
