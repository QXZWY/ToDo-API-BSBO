# ToDo-API-BSBO

## О проекте

**ToDo-API-BSBO** — учебный REST API-сервис для управления задачами, разработанный в рамках дисциплины  
**«Интероперабельность приложений и систем»**.

Проект реализует матрицу приоритетов задач на основе важности и срочности, что позволяет классифицировать задачи по методологии матрицы Эйзенхауэра и эффективно управлять временем.

Сервис предназначен для демонстрации принципов построения REST API, работы с базой данных, реализации аутентификации, валидации данных и модульной архитектуры серверных приложений.

---

## Основной функционал

- Создание, обновление и удаление задач.
- Пометка задач как выполненных.
- Сортировка задач по важности и срочности.
- Получение статистики по задачам.
- Привязка задач к пользователям.
- JWT-аутентификация.
- Разделение API по версиям.

Все маршруты используют общий префикс:

```

/api/v1

```

Модуль статистики доступен по префиксу:

```

/stats

```

---

## Документация API

Swagger-документация доступна по адресу:

```

[http://localhost:8000/docs](http://localhost:8000/docs)

````

---

## Архитектура проекта

- `routers` — маршруты API  
- `models` — модели базы данных  
- `schemas` — Pydantic-схемы  
- `auth_utils` — хеширование паролей и JWT  
- `dependencies` — зависимости FastAPI  
- `database` — подключение к БД  
- `frontend` — клиентская часть  

---

## Развёртывание проекта

### 1. Создание виртуального окружения

В терминале перейдите в папку проекта `ToDo-API-BSBO` и выполните:

```bash
python -m venv venv
````

---

### 2. Активация виртуального окружения

В VS Code, находясь в корневой директории проекта, выполните:

**Windows:**

```bash
venv\Scripts\activate
```

**Linux / macOS:**

```bash
source venv/bin/activate
```

---

### 3. Установка зависимостей

Файл `requirements.txt` содержит список всех необходимых библиотек.

Установите зависимости командой:

```bash
pip install -r requirements.txt
```

---

### 4. Запуск backend-сервера

Запустите сервер FastAPI:

```bash
uvicorn main:app --reload
```

Backend будет доступен по адресу:

```
http://localhost:8000
```

---

## Запуск фронтенда

Перейдите в папку `frontend` и выберите один из вариантов запуска.

---

### Вариант 1: Python

```bash
cd frontend
python -m http.server 3000
```

---

### Вариант 2: Node.js (npx)

```bash
cd frontend
npx http-server -p 3000
```

---

### Вариант 3: VS Code Live Server

1. Установите расширение **Live Server**.
2. Откройте файл `index.html`.
3. Нажмите кнопку **Go Live**.

---

Фронтенд будет доступен по адресу:

```
http://localhost:3000
```
