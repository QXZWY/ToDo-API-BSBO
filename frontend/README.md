
### Запустите фронтенд

Откройте файл `index.html` в браузере. Для правильной работы ES6 модулей рекомендуется использовать локальный сервер:

**Вариант 1: Python**
```bash
cd frontend
python3 -m http.server 3000
```

**Вариант 2: Node.js (npx)**
```bash
cd frontend
npx http-server -p 3000
```

**Вариант 3: VS Code Live Server**
- Установите расширение "Live Server"
- Откройте `index.html` и нажмите "Go Live"

Фронтенд будет доступен по адресу: `http://localhost:3000`
